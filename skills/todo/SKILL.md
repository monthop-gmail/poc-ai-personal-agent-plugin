จัดการรายการสิ่งที่ต้องทำ (TODO) ในไฟล์ ~/personal-agent/notes/TODO.md

ถ้า $ARGUMENTS ว่างเปล่า ให้แสดงรายการ TODO ทั้งหมดที่มีอยู่

ถ้ามี arguments ให้ตีความดังนี้:
- ขึ้นต้นด้วย "เพิ่ม", "add" — เพิ่ม TODO ใหม่ พร้อมวันที่สร้าง
- ขึ้นต้นด้วย "เสร็จ", "done" — เปลี่ยนสถานะเป็น [x] ตาม keyword ที่ระบุ
- ขึ้นต้นด้วย "ลบ", "delete" — ยืนยันกับผู้ใช้ก่อน แล้วลบรายการ
- ขึ้นต้นด้วย "ล้าง", "clean" — ลบรายการที่เสร็จแล้ว [x] ทั้งหมดออก

รูปแบบไฟล์ TODO.md:
```
# รายการสิ่งที่ต้องทำ

## กำลังทำ
- [ ] [2026-03-23] รายการที่ 1
- [ ] [2026-03-23] รายการที่ 2

## เสร็จแล้ว
- [x] [2026-03-22] ~~รายการที่เสร็จแล้ว~~ (เสร็จ: 2026-03-23)
```

หลักการ:
- ใส่วันที่สร้างเสมอ
- เมื่อเสร็จ ย้ายไปหัวข้อ "เสร็จแล้ว" พร้อมวันที่เสร็จ
- แสดงผลเป็นภาษาไทย กระชับ
