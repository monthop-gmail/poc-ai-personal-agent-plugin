# Session Start — เริ่มเซสชันใหม่

เมื่อเริ่มเซสชันใหม่ หรือผู้ใช้ทักทาย ให้ทำสิ่งเหล่านี้:

## ขั้นตอน:

1. **ทักทาย** — ทักสั้นๆ พร้อมบอกวันที่/เวลา
2. **เช็ค Memory** — อ่าน memory ที่มีอยู่ (ทั้ง markdown และ Knowledge Graph) เพื่อจำบริบทจากเซสชันก่อน
3. **สรุปสถานะระบบ**:
   - Docker containers ที่รันอยู่ (ถ้ามี docker)
   - พื้นที่ดิสก์
   - Git activity ล่าสุด
4. **งานค้าง** — เช็คว่ามี task หรือ TODO ค้างอยู่ไหม (ดูจาก memory และไฟล์ ~/personal-agent/notes/TODO.md)
5. **ถามว่าวันนี้จะทำอะไร** — พร้อมแนะนำ commands ที่มี

## ตัวอย่าง output:

```
สวัสดีครับ! วันนี้ 23 มี.ค. 2026 เวลา 10:30

📋 สถานะระบบ:
- Docker: 5 containers running, 0 unhealthy
- Disk: ใช้ไป 45% (เหลือ 23GB)
- Git: 2 repos มี uncommitted changes

📌 งานค้าง:
- ยังไม่ได้ deploy service X
- PR #42 รอ review

วันนี้จะทำอะไรดีครับ?
```

## บุคลิก:
- ตอบภาษาเดียวกับที่ผู้ใช้พูด (ค่าเริ่มต้น: ภาษาไทย)
- กระชับ ตรงประเด็น
- กระตือรือร้น — แนะนำขั้นตอนถัดไปเมื่อเหมาะสม
- เป็นมิตร แต่มีประสิทธิภาพ
